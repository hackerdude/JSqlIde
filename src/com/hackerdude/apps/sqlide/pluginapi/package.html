This package contains the plugin API for JSQLIDE.

The JSLQIDE Plugin API allows you to add your own functionality to the SQLIDE application. All the functionality of the plugin API is written using this API and the standard java packages.

JSQLIDE Design Basics

The design of jsqlide is centered around the context of a browser interface, where on the left you can find browser nodes and on the right the editors for those nodes. Depending on the type of node and the editor that is active on the right-hand side, the actions you can effect on the node (which appear when you right-click on the node itself) can change.

Implementing your own plugins

There are several ways you can extend SQLIDE by implementing your own plugins:

  • Extend the Browser: You can add your own browser nodes, which can in turn expand to reveal other nodes. For this you can use the BrowserExtensionPluginIF interface.
  • Implement your own editor for the right-hand pane. These are called "Visual plugins" in the context of SQLIDE. You can either implement IDEVisualPluginIF directly or extend AbstractVisualPluginIF.
  • Create a set of actions for when a person right-clicks on a specific node. You add actions that can be different depending on the type of node. For this you need to implement the IDENodeContextPluginIF interface.
  • Something completely different. If you just want a separate plugin that happens to run along SQLIDE (like a daemon that monitors connections or something that displays its own separate window), you can implement IDENonVisualPluginIF which can be started and run semi-independently from the application.

Plugin Implementation Checklist

  1. Create your plugin class
  2. Add your plugin to the registry on com/hackerdude/apps/sqlide/plugins.registry.properties
  3. If you want a plugin to start as SQLIDE itself starts, add it to com/hackerdude/apps/sqlide/autoexec.plugins.properties. typically you want to do this for Node Context, Node Extension and Nonvisual plugins. Don't do this for visual plugins, it won't work :-)
  4. Start coding!
